# Quickstart 评测报告

## 概述

- **脚本**：`quickstart.py`
- **目标**：验证 EvalScope 环境，对比不同参数规模 Qwen 模型的推理能力
- **评测方式**：
  1. CLI 等价调用：`Qwen/Qwen2.5-0.5B-Instruct` 在 gsm8k + arc 上各 5 条
  2. 循环对比：0.5B / 1.5B / 3B 在 gsm8k 上各 5 条
